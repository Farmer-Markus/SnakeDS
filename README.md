## SnakeDS
Made in 4 days <br>
Just a small bet I had with a friend

### Building
Set the DEVKITPRO environment variable to your devkitpro installation, then run
```
mkdir build
cd build
cmake ..
make
```
This should build the .nds rom in a few seconds
